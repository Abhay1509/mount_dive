# Trekking Website Starter

## Backend (Node.js + Express + MongoDB)
- Run with: 
```bash
cd backend
npm init -y
npm install express mongoose cors
node server.js
```

## Frontend (React)
- Run with:
```bash
cd frontend
npx create-react-app .
npm install
npm start
```

Replace src/App.js with provided App.jsx and src/index.js with given index.js
